## Overview
This project is an **Expenses Analysis Dashboard** built using **Power BI** to track, analyze, and visualize organizational expenses across months, voucher types, and expense categories.  
The goal is to convert raw financial transaction data into **clear, decision-ready insights**.

This dashboard helps answer:
- Where is money being spent the most?
- How do expenses trend month over month?
- Which expense categories and voucher types drive total costs?

## Dashboard Features
### 1. Expenses by Month
- Monthly expense trend from **April to December**
- Identifies peak spending months (October is the highest)
- Helps monitor cost fluctuations over time

### 2. Expenses by Voucher Type
- Breakdown of expenses across different voucher types
- Highlights which transaction types contribute most to total expenses
- Useful for audit and accounting-level analysis

### 3. Expenses by Particulars
- Category-level expense analysis
- Major cost drivers include:
  - Petrol & Diesel
  - Lease Rent
  - Freight Charges
  - Loan Interest
- Enables targeted cost optimization

### 4. Interactive Filters
- Filter by **Month**
- Filter by **Expense Category (Particulars)**
- Enables focused and dynamic analysis

## Business Insights
- A small number of expense categories contribute to the majority of total costs (Pareto effect).
- Fuel, rent, and freight charges are major recurring expenses.
- Seasonal or operational spikes are visible in monthly trends.
- Helps management identify areas for **cost control and budgeting**.
